# PPO

## Название проекта

Приложение для детективного агентства.

## Краткое описание идеи проекта

Разработать приложение для детективного агентства которое позволяет: добавлять записи о людях, информацию о них, связи
между людьми и тд. Визуализировать связи между ними, передвижения.

## Краткое описание предметной области

Предметной областью является информация о людях.

## Краткое обоснование целесообразности и актуальности проекта

Удобное приложение для сотрудников и клиентов детективного агентства должно облегчить работу.

## Краткое описание акторов (ролей)

1. Сотрудник агентства(может читать, изменять информацию)
2. Старший сотрудник(создает аккаунты)
3. Госслужащий/человек с особыми правами(читает всю информацию свободно, менять не может)

## Пользовательские сценарии;

Неавторизованный пользователь может:

1. Авторизоваться

Сотрудник может:

1. Добавлять информацию о людях (связанные места, перемещения, связи с другими людьми и тд)
2. Удалять информацию (если ошибочная)
3. Получать информацию

Старший сотрудник может:

1. Все, что может сотрудник.
2. Создавать аккаунты.
